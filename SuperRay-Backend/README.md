<h1>Back-end Server for Super Ray</h1>
<h3>Clone and create a virtual environment</h3>
<h5>Open terminal and enter </h5>
<ul>
  <li>python -m venv env</li>
  <li>pip install -r requirments.txt</li>
  <li>python manage.py runserver</li>
</ul>
